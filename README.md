This project demonstrates how Continuous Integration and Continuous Deployment (CI/CD) can be achieved using GitHub Actions, Ansible, and AWS.  
It automates the build, test, and deployment process, ensuring faster delivery, consistency, and scalability across environments.

Tools & Technologies

- GitHub Actions – for CI/CD automation
- Ansible – for configuration management
- AWS EC2 – for hosting and deployment
- Docker – for containerization

Features

- Automated build and deployment pipeline
- Continuous monitoring and error reporting
- Scalable and reliable infrastructure setup
- Cost-efficient cloud usage (AWS Free Tier)

Benefits

- Saves time through automation
- Reduces human error
- Ensures consistency across environments
- Enables faster feedback and delivery
